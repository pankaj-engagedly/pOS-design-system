Save to KB — Bookmarklet

Save any web page to your pOS Knowledge Base with one tap. Works on iOS Safari, iOS Chrome, Android, and desktop browsers. No extension needed.

Find this in pOS → Settings → API Keys. The key stays in your bookmark — never sent to us.

Your Bookmarklet

This button saves the current page to your KB. Set it up once, use it forever.

Long-press the blue "Save to pOS KB" button above and select "Copy Link"
Bookmark any page in Safari (tap Share → Add Bookmark)
Go to Bookmarks (book icon) → find the bookmark → Edit
Clear the URL field and paste the copied link
Rename it to "Save to KB" and tap Done
Done! Now when browsing any article, tap the bookmark to save it to KB
Copy the bookmarklet code below (tap the Copy button)
Bookmark any page in Chrome (⋯ menu → Bookmark)
Go to Bookmarks (⋯ → Bookmarks) → find it → tap Edit
Clear the URL and paste the copied code
Rename to "Save to KB" and save
To use: type "Save" in the address bar → tap the bookmark suggestion
Simply drag the blue "Save to pOS KB" button to your bookmarks bar
Done! Click it on any page to save to KB
