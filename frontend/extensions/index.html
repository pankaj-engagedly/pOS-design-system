pOS Knowledge Base — Chrome Extension

Right-click any page or use the toolbar icon to save URLs, articles, and videos to your KB with tags and collections.

Download Extension (.zip)

Installation (Chrome / Brave / Edge)

  1. Download and unzip the file
  2. Open chrome://extensions in your browser
  3. Enable Developer mode (toggle in top-right)
  4. Click "Load unpacked"
  5. Select the unzipped chrome-kb-saver-prod folder
  6. Click the extension icon in the toolbar and sign in with your pOS account
